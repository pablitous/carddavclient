# carddavclient
CardDav Client in php+js+css+ajax

This is working for baikal server


Edit the config.php in ./clientajax/config.php

Enter your server.

Try it.

You need:

· php 5.4+ and CURL

· install http://sabre.io/dav/install/ or just unrar the sabredav.rar in the same folder.
